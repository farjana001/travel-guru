# Travel Guru
A Travel plan booking web application. 

[Live Website](https://travel-gurus.web.app)


### Features 
* Clickable travel place card to book hotels.
* Booking Form.
* Login with Email and Password.
* Firebase authentication.

### Technology
* React
* React Router
* React Hook Form
* Javascript
* Bootstrap
* Material UI
* HTML
* Css
